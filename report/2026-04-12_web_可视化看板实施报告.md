# 数字生命内部状态 Web 可视化实施报告

> **日期**: 2026-04-12  
> **分支**: `feat/life-engine-web-viz`  
> **状态**: ✅ 已集成

---

## 1. 概览

为了提升 Neo-MoFox 数字生命在运行时的透明度与“可理解性”，我们在 `life_engine` 插件中集成了一套高性能、高审美的 Web 可视化工具集。这套工具不依赖重型的现代前端框架（如 React/Vue），而是通过 `FastAPI` 挂载路由，结合 `HTML5 Canvas` 和 `D3.js / Force-Graph` 实现极轻量且视觉效果拔群的实时监控。

---

## 2. 核心功能

### 2.1 SNN 皮层下驱动看板 (`/snn`)

- **实时动力学渲染**：使用原生 Canvas 渲染 8→16→6 的脉冲神经网络。
- **STDP 脉冲粒子系统**：
    - 前向传播的脉冲以白色粒子形式沿突触路径流动。
    - 粒子密度与流速动态反映了突触权重的强度。
- **驱动态监控**：通过磨砂玻璃效果的侧边栏，实时观察 6 维驱动信号（如唤醒度、倾向性等）的平滑演变。

### 2.2 仿生记忆图谱 (`/memory`)

- **关联力导向图**：将记忆数据库中的文件 (File) 与概念 (Concept) 节点化，边宽度代表关联权重。
- **激活扩散模拟 (Ignition)**：
    - 用户可点击任意记忆节点，手动触发“激发联想 (Spread Activation)”。
    - 系统会后台计算并在前端高亮展示激活扩散的完整路径，直观展示 AI 的“心流”与“联想过程”。
- **动态遗忘观测**：节点大小随 `activation_strength` 动态调整，直观体现记忆的衰减与增强。

---

## 3. 技术实现

### 3.1 后端 (Python/FastAPI)

- **`snn_router.py`**:
    - `/api/state`: 以 1Hz 频率提供神经网络的全量快照（膜电位、权重、脉冲痕迹）。
- **`memory_router.py`**:
    - `/api/graph`: 按活跃度排序拉取 top-K 记忆节点及其内部拓扑。
    - `/api/activate`: 接受种子节点 ID，调用 `MemoryService.spread_activation` 返回传播路径。

### 3.2 前端 (HTML/Canvas/D3)

- **深色美学设计**：Slate 900+ 底色，配合荧光蓝 (Cyan) 与霓虹粉 (Pink) 的配色方案。
- **力导向物理仿真**：使用 `force-graph` 库处理节点排布，支持平滑缩放与拖拽。
- **粒子排放引擎**：在 SNN 画布上实现高频率粒子发射，模拟神经信号。

---

## 4. 访问与配置

- **访问入口**：
    - SNN: `/router/snn/`
    - 记忆图谱: `/router/memory/`
- **挂载机制**：通过 `LifeEnginePlugin.get_components` 自动注册为 `BaseRouter` 组件，由主应用自动发现并挂载到 HTTP Server。
