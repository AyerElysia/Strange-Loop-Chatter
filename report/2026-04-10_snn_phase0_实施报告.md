## 1. 目标回顾

将 Neo-MoFox 的 life_engine 从"轮次驱动"的 LLM 附属系统，升级为具有**连续内在状态**的数字生命体。核心手段：在 life_engine 中嵌入一个纯 numpy 实现的 SNN（脉冲神经网络）驱动核，作为系统的"皮层下"状态层。

**关键设计原则**：
- LLM = 大脑皮层（认知推理），SNN = 皮层下系统（持续存在）
- SNN 有独立于 LLM 心跳的 tick 循环——心跳之间系统仍在"活"
- STDP 学习使系统本身具备适应能力，不依赖外部训练
- 所有功能默认关闭，可一键回退

## 4. 架构设计

### 4.1 网络结构

```
输入层 (8维)              隐藏层 (16 LIF)           输出层 (6 LIF)
┌──────────────┐    ┌──────────────────┐    ┌───────────────┐
│ msg_in       │    │                  │    │ arousal       │
│ msg_out      │    │  tau=15.0        │    │ valence       │
│ tool_success │───→│  threshold=0.3   │───→│ social_drive  │
│ tool_fail    │    │  STDP 突触       │    │ task_drive    │
│ idle_beats   │    │  lr+=0.01        │    │ exploration   │
│ tell_dfc     │    │  lr-=0.005       │    │ rest_drive    │
│ new_content  │    │                  │    │               │
│ silence      │    │  16 个 LIF 神经元 │    │ 6 个 LIF 神经元│
└──────────────┘    └──────────────────┘    └───────────────┘
     ↑                                            │
     │              奖赏信号                       │ EMA 平滑
     └──── 事件流 ──────────────────────────────────┘
```

### 4.2 数据流

```
1. SNN 独立 tick (每10秒)：零输入步进 → 膜电位自然衰减
2. 心跳前 (heartbeat_pre)：事件→8维特征 → SNN.step(features, last_reward)
3. LLM 心跳执行：（可选）SNN 驱动状态注入 prompt
4. 心跳后 (heartbeat_post)：工具调用统计→奖赏信号 → 存储供下轮使用
5. 定期持久化：SNN 全部状态（权重/膜电位/痕迹）→ JSON
```

### 4.3 安全机制

```
                 enabled=false ──→ SNN 完全不初始化
                 enabled=true
                       │
                 shadow_only=true ──→ SNN 运行但不注入任何内容
                 shadow_only=false
                       │
                 inject_to_heartbeat=false ──→ SNN 运行，状态可查但不注入
                 inject_to_heartbeat=true ──→ "内在驱动态：激活高、社交中..." 注入 prompt
```

## 6. 关键设计决策

### 6.1 为什么降低阈值 (0.8 → 0.3)

初始阈值 0.8 配合 Xavier 初始化权重（~0.4 scale），导致正常输入无法触发隐藏层脉冲，STDP 学习完全失效。将阈值降至 0.3 后，隐藏层能产生合理的脉冲率，验证了学习通路。

### 6.2 为什么用 EMA 而不是脉冲率

LIF 神经元的输出是离散脉冲（0/1），直接用作驱动信号会导致剧烈震荡。使用膜电位的 EMA（α=0.1）提供更平滑的连续信号，适合注入 prompt。

### 6.3 为什么 SNN 有独立 tick

如果 SNN 只在心跳时更新（每 3-10 分钟），那么心跳间隔内的时间流逝不会被编码。独立 tick（10s）确保即使系统空闲，SNN 的膜电位仍在衰减。这是"连续存在"的物理基础。
