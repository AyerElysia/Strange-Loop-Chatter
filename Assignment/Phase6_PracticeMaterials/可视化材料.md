### 2.2 STDP学习过程可视化

```
STDP学习过程：

事件流 → 特征提取 → SNN输入 → Reward信号 → STDP权重调整

示例：工具成功场景

事件流：
  tool_call → tool_result(success=True)
  
特征提取：
  features = [0.0, 0.0, 0.5, 0.85, ...]
  tool_success_rate = 0.85
  
SNN输入：
  input_spikes = [0, 0, 0.5, 0.85, ...]
  
SNN前向传播：
  hidden_spikes = [0.45, 0.52, 0.48, 0.50]
  output_rates = [0.50, 0.50, 0.55, 0.60, ...]
  
Reward计算：
  tool_success_count = 1
  reward = 0.3 + 0.15 = 0.45（正reward）
  
STDP权重调整：
  LTP（长时程增强）：
    pre_activity = 0.85
    post_activity = 0.52
    Δw = lr * pre_activity * post_activity * reward
    syn_in_hid.W[2][1] += Δw = 0.50 → 0.55
  
  结果：
    强化特定神经元路径（16维隐藏层中的特定神经元）
    SNN学习：工具成功 → 强化特定驱动路径

长期效果：
  100次工具成功后：
    syn_in_hid.W[2][1] = 0.50 → 0.70（强强化）
    SNN学习到：工具成功 → 强化社交驱动
```
