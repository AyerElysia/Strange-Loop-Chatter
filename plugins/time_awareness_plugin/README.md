# Time Awareness Plugin - 时间感知插件

**版本：** 1.0.0
**作者：** Neo-MoFox Team

## 功能介绍

时间感知插件为爱莉希雅提供时间感知能力，让她能够：

- **感知当前时间** - 知道现在是几年几月几日、什么时辰
- **中式时间描述** - 使用子丑寅卯等传统时间单位
- **主动查询时间** - 通过 query_time 工具获取准确时间
- **时间合适问候** - 根据当前时间给出恰当的问候和回应

## 时间格式示例

```
2026 年 3 月 18 日 (周三) 酉时 (傍晚，17 点 30 分，4 刻)，马年
```

包含：
- 公历日期（年/月/日 + 星期）
- 中式时辰（酉时）
- 时间段描述（傍晚）
- 现代时间（17 点 30 分）
- 刻数（4 刻）
- 生肖年份（马年）

## 使用场景

### 1. 时间相关问候

```
用户：早
爱莉希雅：query_time() → "现在是 2026 年 3 月 18 日 酉时 (傍晚...)"
爱莉希雅：现在是傍晚哦，应该说晚上好～这么晚了还没休息吗？
```

### 2. 时间推理

```
用户：这么晚了你还在线啊
爱莉希雅：query_time() → 看到现在是戌时 (晚上 7-9 点)
爱莉希雅：嘿嘿，我可是夜猫子呢～而且我想陪你聊天呀
```

### 3. 日记引用

```
用户：今天早上我...
爱莉希雅：query_time() → 现在是下午 3 点
爱莉希雅：哦？今天早上发生了什么？让我查查日记...
```

## 安装配置

### 1. 启用插件

在 `config/core.toml` 中添加：

```toml
[bot]
plugins = ["time_awareness_plugin"]
```

### 2. 自定义配置（可选）

编辑 `config/plugins/time_awareness_plugin/config.toml`：

```toml
[settings]
enabled = true          # 启用插件
inject_on_load = true   # 加载时自动注入时间
auto_refresh = false    # 自动刷新（暂未实现）

[format]
use_chinese_shichen = true  # 使用中式时辰
use_zodiac = true           # 显示生肖
use_ke = true               # 显示刻
custom_format = ""          # 自定义格式（空则使用默认）
```

### 3. 重启 Bot

```bash
python -m src.app.main
```

## 工具接口

### query_time 工具

**工具名称：** `query_time`

**功能描述：** 查询当前时间

**参数：** 无

**返回值：**
```json
{
    "current_time": "2026 年 3 月 18 日 (周三) 酉时 (傍晚，17 点 30 分，4 刻)，马年",
    "reminder": "时间已查询。现在你可以根据当前时间给出合适的问候或回应了。"
}
```

## 技术原理

### System Reminder 注入

插件在 `on_plugin_loaded` 钩子中使用 `add_system_reminder` 注入时间：

```python
add_system_reminder(
    bucket="actor",
    name="current_datetime",
    content=f"现在是 {time_str}。在回复用户时，请结合当前时间给出合适的问候和回应。",
)
```

**为什么用 system reminder：**
- 与人格提示词分离，结构清晰
- 可以随时刷新（未来功能）
- 不依赖模板渲染

## 最佳实践

### 好的时间使用示例

✅ 结合时间给出合适问候：
```
query_time() → 卯时 (清晨)
"早上好呀！这么早就起床了？"
```

✅ 用时间信息丰富对话：
```
query_time() → 戌时 (晚上 8 点)
"都这个点了，你还没吃晚饭吗？"
```

### 不好的时间使用示例

❌ 机械复述时间：
```
query_time() → 2026 年 3 月 18 日...
"现在是 2026 年 3 月 18 日戌时..."（无意义重复）
```

❌ 忽略时间信息：
```
query_time() → 深夜 2 点
用户：早
爱莉希雅：早啊～（完全忽略时间是深夜）
```

## 故障排除

### 问题：时间不更新

**现象：** Bot 运行很久后，时间仍然是启动时的时间

**原因：** 当前版本只在插件加载时注入一次时间

**解决方法：**
1. 重启 Bot 刷新时间
2. 等待后续版本实现 auto_refresh 功能
3. 爱莉希雅可以使用 query_time 工具获取最新时间

### 问题：爱莉希雅不使用时间信息

**可能原因：** LLM 没有注意到 time reminder

**解决方法：**
1. 检查插件是否已加载
2. 等待模型自适应（多次对话后可能改善）
3. 在对话中引导她注意时间
