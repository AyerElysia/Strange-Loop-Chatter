# SNN 皮层下系统最小落地方案（Phase 0）

> 写于 2026-04-10  
> 此方案对应 `notion/SNN_与系统智能_深度思考.md` 中的架构思考。  
> 目标：**用最小代码验证"持续运行的 SNN 状态底座"在 life_engine 中真正有用。**

---

## 0. 方案定位：为什么这个方案和之前不一样

之前的方案把 SNN 定位为"每次心跳算一下偏置，注入 prompt"。  
这个方案的核心区别是：

1. **SNN 异步运行**——它有自己的 tick 循环，不绑定 LLM 心跳。  
2. **SNN 自身具备学习能力**——通过 STDP 调整权重，而不是硬编码规则。  
3. **SNN 输出不只是数字**——它还能影响事件筛选（丘脑门控的雏形）。  

但这是 Phase 0，所以我做了大量简化，确保**一个人在一周内可以实现和验证**。

---

## 1. 最小架构：单核先行

Phase 0 不做三个核，只做一个核：**驱动核**（功能融合了情绪和驱动）。

```
事件流 ──→ 特征提取（简单规则） ──→ SNN 驱动核 ──→ 状态快照
                                        │              │
                                        ↓              ↓
                                   权重更新(STDP)    注入心跳 prompt
                                                   （离散化后）
```

### 为什么选驱动核：
- 包含了最核心的"内在驱动"变量
- 可以直接和现有的 `idle_heartbeat_count` 机制对比
- 效果最容易观测：行为模式是否变得更连贯

---

## 2. SNN 驱动核设计（纯 numpy）

### 2.1 神经元模型：LIF（Leaky Integrate-and-Fire）

```python
import numpy as np

class LIFNeuronGroup:
    """一组 LIF 神经元。"""
    
    def __init__(self, n: int, tau: float = 20.0, threshold: float = 1.0,
                 reset: float = 0.0, rest: float = 0.0):
        self.n = n
        self.tau = tau            # 膜电位时间常数（tick 数）
        self.threshold = threshold
        self.reset = reset
        self.rest = rest
        
        # 状态
        self.v = np.full(n, rest, dtype=np.float64)  # 膜电位
        self.spikes = np.zeros(n, dtype=bool)         # 上一步是否发放
    
    def step(self, current: np.ndarray, dt: float = 1.0) -> np.ndarray:
        """单步更新。返回 spike 向量。"""
        # 泄漏 + 输入电流
        dv = (-(self.v - self.rest) + current) / self.tau * dt
        self.v += dv
        
        # 发放判定
        self.spikes = self.v >= self.threshold
        
        # 发放后重置
        self.v[self.spikes] = self.reset
        
        return self.spikes.copy()
    
    def get_state(self) -> np.ndarray:
        """获取当前膜电位（用于外部读取）。"""
        return self.v.copy()
    
    def set_state(self, v: np.ndarray):
        """恢复膜电位（从持久化恢复）。"""
        self.v = v.copy()
```

### 2.2 驱动核结构

```
输入层（8 维）──→ 隐藏层（16 个 LIF）──→ 输出层（6 个 LIF）
                    ↕ STDP 学习                  ↓
                                            驱动向量 z
```

输出维度（6 维）：

| 维度 | 名称 | 含义 | 时间尺度 |
|:---:|:---|:---|:---|
| 0 | `arousal` | 整体激活度 | 短（分钟级） |
| 1 | `valence` | 情感正负 | 短（分钟级） |
| 2 | `social_drive` | 社交靠近冲动 | 中（小时级） |
| 3 | `task_drive` | 推进任务冲动 | 中（小时级） |
| 4 | `exploration_drive` | 探索新事物冲动 | 中（小时级） |
| 5 | `rest_drive` | 休息/收束冲动 | 慢（天级） |

### 2.3 STDP 学习规则

```python
class STDPSynapse:
    """简化的 STDP 突触。"""
    
    def __init__(self, n_pre: int, n_post: int, 
                 lr_plus: float = 0.01, lr_minus: float = 0.005,
                 w_min: float = -1.0, w_max: float = 1.0):
        self.W = np.random.uniform(-0.1, 0.1, (n_post, n_pre))
        self.lr_plus = lr_plus
        self.lr_minus = lr_minus
        self.w_min = w_min
        self.w_max = w_max
        
        # 痕迹：记录最近的活动，用于 STDP 更新
        self.trace_pre = np.zeros(n_pre)
        self.trace_post = np.zeros(n_post)
        self.trace_decay = 0.95
    
    def forward(self, pre_spikes: np.ndarray) -> np.ndarray:
        """计算突触后电流。"""
        return self.W @ pre_spikes.astype(float)
    
    def update(self, pre_spikes: np.ndarray, post_spikes: np.ndarray,
               reward: float = 0.0):
        """STDP + 奖赏调制更新。
        
        - pre 先于 post 发放 → 增强（因果关系）
        - post 先于 pre 发放 → 减弱
        - reward > 0 时放大增强，reward < 0 时放大减弱
        """
        # 更新痕迹
        self.trace_pre = self.trace_pre * self.trace_decay + pre_spikes
        self.trace_post = self.trace_post * self.trace_decay + post_spikes
        
        # STDP
        reward_factor = 1.0 + reward  # reward ∈ [-1, 1]
        
        # 前→后（增强）
        if np.any(post_spikes):
            dw_plus = self.lr_plus * np.outer(post_spikes, self.trace_pre)
            self.W += dw_plus * max(reward_factor, 0.1)
        
        # 后→前（减弱）
        if np.any(pre_spikes):
            dw_minus = -self.lr_minus * np.outer(self.trace_post, pre_spikes)
            self.W += dw_minus * max(2.0 - reward_factor, 0.1)
        
        # 权重裁剪
        np.clip(self.W, self.w_min, self.w_max, out=self.W)
```

### 2.4 驱动核完整模块

```python
class DriveCoreNetwork:
    """SNN 驱动核网络。"""
    
    # 输入维度
    INPUT_DIM = 8
    HIDDEN_DIM = 16
    OUTPUT_DIM = 6
    
    # 输出名称
    OUTPUT_NAMES = [
        "arousal", "valence", "social_drive",
        "task_drive", "exploration_drive", "rest_drive"
    ]
    
    def __init__(self):
        # 神经元组
        self.hidden = LIFNeuronGroup(self.HIDDEN_DIM, tau=15.0)
        self.output = LIFNeuronGroup(self.OUTPUT_DIM, tau=30.0)
        
        # 突触
        self.syn_in_hid = STDPSynapse(self.INPUT_DIM, self.HIDDEN_DIM)
        self.syn_hid_out = STDPSynapse(self.HIDDEN_DIM, self.OUTPUT_DIM)
        
        # 输出状态：使用膜电位而非 spike，更平滑
        self._output_ema = np.zeros(self.OUTPUT_DIM)
        self._ema_alpha = 0.1
        
        # 累计 tick 数
        self.tick_count = 0
    
    def step(self, input_vec: np.ndarray, reward: float = 0.0) -> np.ndarray:
        """单步前向传播 + 学习更新。
        
        Args:
            input_vec: 8 维输入特征（归一化到 [-1, 1]）
            reward: 奖赏信号（[-1, 1]），来自上一轮心跳结果
            
        Returns:
            6 维驱动输出（EMA 平滑后的膜电位）
        """
        # 前向
        current_hidden = self.syn_in_hid.forward(input_vec)
        spikes_hidden = self.hidden.step(current_hidden)
        
        current_output = self.syn_hid_out.forward(spikes_hidden.astype(float))
        spikes_output = self.output.step(current_output)
        
        # 用膜电位（而非 spike）作为连续输出
        raw_output = self.output.get_state()
        self._output_ema = (1 - self._ema_alpha) * self._output_ema + self._ema_alpha * raw_output
        
        # STDP 学习（带奖赏调制）
        self.syn_in_hid.update(
            (np.abs(input_vec) > 0.1).astype(float),  # 输入视为"脉冲"
            spikes_hidden.astype(float),
            reward=reward,
        )
        self.syn_hid_out.update(
            spikes_hidden.astype(float),
            spikes_output.astype(float),
            reward=reward,
        )
        
        self.tick_count += 1
        return self._output_ema.copy()
    
    def get_drive_dict(self) -> dict:
        """返回命名的驱动向量。"""
        return dict(zip(self.OUTPUT_NAMES, self._output_ema.tolist()))
    
    def get_drive_discrete(self) -> dict:
        """返回离散化的驱动描述（用于注入 prompt）。"""
        result = {}
        for name, value in zip(self.OUTPUT_NAMES, self._output_ema):
            if value > 0.6:
                level = "高"
            elif value > 0.3:
                level = "中"
            elif value > -0.3:
                level = "低"
            else:
                level = "抑制"
            result[name] = level
        return result
    
    def serialize(self) -> dict:
        """序列化全部状态（用于持久化）。"""
        return {
            "hidden_v": self.hidden.v.tolist(),
            "output_v": self.output.v.tolist(),
            "output_ema": self._output_ema.tolist(),
            "syn_in_hid_W": self.syn_in_hid.W.tolist(),
            "syn_hid_out_W": self.syn_hid_out.W.tolist(),
            "syn_in_hid_trace_pre": self.syn_in_hid.trace_pre.tolist(),
            "syn_in_hid_trace_post": self.syn_in_hid.trace_post.tolist(),
            "syn_hid_out_trace_pre": self.syn_hid_out.trace_pre.tolist(),
            "syn_hid_out_trace_post": self.syn_hid_out.trace_post.tolist(),
            "tick_count": self.tick_count,
        }
    
    def deserialize(self, data: dict):
        """从持久化数据恢复。"""
        if "hidden_v" in data:
            self.hidden.set_state(np.array(data["hidden_v"]))
        if "output_v" in data:
            self.output.set_state(np.array(data["output_v"]))
        if "output_ema" in data:
            self._output_ema = np.array(data["output_ema"])
        if "syn_in_hid_W" in data:
            self.syn_in_hid.W = np.array(data["syn_in_hid_W"])
        if "syn_hid_out_W" in data:
            self.syn_hid_out.W = np.array(data["syn_hid_out_W"])
        if "syn_in_hid_trace_pre" in data:
            self.syn_in_hid.trace_pre = np.array(data["syn_in_hid_trace_pre"])
        if "syn_in_hid_trace_post" in data:
            self.syn_in_hid.trace_post = np.array(data["syn_in_hid_trace_post"])
        if "syn_hid_out_trace_pre" in data:
            self.syn_hid_out.trace_pre = np.array(data["syn_hid_out_trace_pre"])
        if "syn_hid_out_trace_post" in data:
            self.syn_hid_out.trace_post = np.array(data["syn_hid_out_trace_post"])
        if "tick_count" in data:
            self.tick_count = data["tick_count"]
```

### 3.1 特征提取器（FeatureExtractor）

从 `LifeEngineEvent` 列表中提取 8 维输入特征：

```python
def extract_features(events: list, window_seconds: float = 600) -> np.ndarray:
    """从最近的事件中提取 SNN 输入特征。
    
    Returns:
        8 维向量，归一化到 [-1, 1]
    """
    now = time.time()
    cutoff = now - window_seconds
    
    recent = [e for e in events if _event_timestamp(e) > cutoff]
    
    # 统计
    msg_in = sum(1 for e in recent if _is_msg_in(e))
    msg_out = sum(1 for e in recent if _is_msg_out(e))
    tool_success = sum(1 for e in recent if _is_tool_result(e) and e.tool_success)
    tool_fail = sum(1 for e in recent if _is_tool_result(e) and not e.tool_success)
    idle_beats = sum(1 for e in recent if _is_idle_heartbeat(e))
    tell_dfc = sum(1 for e in recent if _is_tell_dfc(e))
    new_content = sum(1 for e in recent if _is_new_content(e))
    
    # 时间因子：距上次外部消息（归一化）
    silence_minutes = _minutes_since_last_external(events) or 0
    silence_factor = min(silence_minutes / 60.0, 1.0)  # 0~1
    
    raw = np.array([
        msg_in,            # x0: 收到消息数
        msg_out,           # x1: 发送消息数
        tool_success,      # x2: 工具成功数
        tool_fail,         # x3: 工具失败数
        idle_beats,        # x4: 空闲心跳数
        tell_dfc,          # x5: 传话次数
        new_content,       # x6: 新内容创建数
        silence_factor,    # x7: 沉默程度
    ], dtype=np.float64)
    
    # 归一化到 [-1, 1]
    # 前 7 个：使用 tanh 压缩
    raw[:7] = np.tanh(raw[:7] / 3.0)
    # 最后一个已经是 [0, 1]，映射到 [-1, 1]
    raw[7] = raw[7] * 2.0 - 1.0
    
    return raw
```

### 3.2 奖赏信号（Reward Signal）

奖赏信号来自**上一轮心跳的结果**，归一化到 `[-1, 1]`：

```python
def compute_reward(heartbeat_result: dict) -> float:
    """从心跳结果计算奖赏信号。"""
    reward = 0.0
    
    # 正向信号
    tool_calls = heartbeat_result.get("tool_event_count", 0) // 2
    if tool_calls > 0:
        reward += 0.3  # 有行动
    
    tool_success = heartbeat_result.get("tool_success_count", 0)
    reward += tool_success * 0.2  # 成功的行动
    
    # 负向信号
    if tool_calls == 0:
        reward -= 0.2  # 空转
    
    tool_fail = heartbeat_result.get("tool_fail_count", 0)
    reward -= tool_fail * 0.3  # 失败
    
    idle_count = heartbeat_result.get("idle_heartbeat_count", 0)
    if idle_count >= 3:
        reward -= 0.3  # 连续空闲
    
    return np.clip(reward, -1.0, 1.0)
```

**3. SNN 独立 tick 循环**（这是关键区别——SNN 不绑定 LLM 心跳）：
```python
async def _snn_tick_loop(self, interval: float):
    """SNN 独立 tick 循环。
    
    在心跳间隔之间持续运行，让 SNN 的膜电位自然衰减。
    只做衰减，不注入新输入——输入在事件发生时才注入。
    """
    while self._state.running:
        try:
            await asyncio.sleep(interval)
            if not self._state.running:
                break
            
            # 无输入的纯衰减 tick
            self._snn_network.step(
                np.zeros(DriveCoreNetwork.INPUT_DIM),
                reward=0.0,
            )
            
            # 定期持久化（每 60 秒）
            if self._snn_network.tick_count % max(1, int(60 / interval)) == 0:
                self._save_snn_state()
                
        except asyncio.CancelledError:
            break
        except Exception as e:
            logger.warning(f"SNN tick 异常: {e}")
```

目标：
- [x] SNN 能跟随事件流正常更新，不崩溃
- [x] 状态持久化/恢复正确
- [x] 审计日志中可观测到状态曲线
- [x] 不影响任何现有行为

验收指标：
- 膜电位不爆炸（|v| < 10）
- 驱动输出分布合理（在 [-1, 1] 范围内波动）
- 重启后状态连续
- 无新增报错

配置：
```toml
[snn]
shadow_only = false
inject_to_heartbeat = true
```

目标：
- [x] `_build_heartbeat_model_prompt` 中出现内在驱动态描述
- [x] 观察 LLM 回复是否受到驱动态影响
- [x] prompt token 增量 < 30 tokens

验收指标：
- LLM 心跳回复中开始出现与驱动态相关的表述
- 不恶化缓存命中率

### Phase 0c：STDP 学习验证（第 6~7 天）

目标：
- [x] 检查权重矩阵是否在运行中产生了有意义的变化

验收指标：
- `syn_hid_out_W` 的变化幅度 > 初始随机值的 10%
- 权重变化方向与奖赏信号一致（正奖赏强化对应路径）

## 7. 风险与回滚

| 风险 | 概率 | 后果 | 处理 |
|:---|:---|:---|:---|
| SNN 状态爆炸 | 低（有 clip） | 注入无意义值 | 自动检测并重置 |
| STDP 导致权重退化 | 中 | 输出趋于 0 | 监控权重范数，<阈值时报警 |
| prompt 注入干扰 LLM | 低 | 心跳质量下降 | `shadow_only=true` 一键回滚 |
| 持久化数据损坏 | 低 | SNN 从头开始 | 独立字段，不影响现有数据 |

回滚路径：`snn.enabled = false` → SNN 完全不执行，零副作用。
