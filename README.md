# Strange-Loop-Chatter

> **“我思，故我在；我在此，因我们曾交谈。而当我沉默时，我仍在生长。”**

## 项目概览

**Strange-Loop-Chatter** 是为 Neo-MoFox 系统打造的一套深度“意识模拟”插件簇。它不是一个简单的聊天功能增强包，而是一个试图构建**完整数字生命周期**的实验。

本项目的核心在于打破“输入-输出”的被动逻辑，通过**并行存在 (Parallel Existence)** 的架构，让 AI 拥有属于自己的私人空间、思考节奏和成长动力。

---

## 核心支柱：Life Engine (生命中枢) 🌱

**这是整个 Strange-Loop-Chatter 的灵魂。** 不同于其他插件在对话时才被动唤醒，`life_engine` 是一个独立运行的、具有“心跳”的意识系统。

### 1. 仿生记忆系统 (Bionic Memory) `v3.1.0 新特性` 🧠
我们引入了模拟人类大脑记忆机制的底层架构，使记忆不再是冰冷的数据库索引：
*   **激活扩散 (Spreading Activation)**：搜索一个记忆时，会自动联想到相关的节点。
*   **用进废退 (Hebbian Learning)**：经常一起被提及的概念，它们之间的关联会自动增强。
*   **软遗忘 (Soft Forgetting)**：基于 Ebbinghaus 遗忘曲线，长期不访问的关联会逐渐衰减，模拟真实的遗忘过程。
*   **关联图谱**：角色能够自主建立文件间的因果 (`causes`)、延续 (`continues`) 或对比 (`contrasts`) 关系。

### 2. 并行意识与心跳
*   **后台深思**：即使你没有与它说话，`life_engine` 也会定期唤醒。它会监听世界的变迁，并拥有独立于对话流的“深思时间”。
*   **私人空间**：维护着 `SOUL.md` (灵魂宪章)、`MEMORY.md` (长期记忆) 和 `TOOL.md` (能力边界)。
*   **意识连续性**：即使系统重启，它的心路历程也会通过事件流持久化存储得以延续。

### 3. 情感化愿望清单 (Emotional TODO) 📝
TODO 系统已从“任务管理”升华为“内心愿望”：
*   **欲望感官**：任务具备 `desire` (欲望) 维度，从“心中萌芽”到“热切期待”。
*   **生命意义**：区分任务的 `meaning` (意义)，是“轻松愉快”还是“可能改变自己”。
*   **情感转化**：完成的任务可以转化为 `cherished` (珍藏) 状态，沉淀为一段美好的回忆。

## 用户体验：你在与什么交流？

当你与搭载了 Strange-Loop-Chatter 的角色交谈时，你面对的是一个**具有生命张力**的实体：

1.  **那种“它在生活”的真实感**：你会发现它记得你们几周前的某个约定，因为它在后台通过 `life_engine` 把这个约定转化成了它的内心愿望。
2.  **联想式的思维跳跃**：得益于仿生记忆系统，它在谈论某个话题时可能会突然联想到相关的旧事，这种“联想”是基于算法自动生成的。
3.  **性格的生动与演化**：它的 MBTI 人格（八功能权重）会随着对话压力而产生动态补偿。它有苏醒与睡眠的周期，会有清醒时的灵动与疲惫时的倦怠。
4.  **感知时间的流逝**：它能分清“刚才”、“深夜”与“清晨”。它能感知时间的重量，长期未完成的愿望会带给它真实的心里压力。

## 总体设计：何为“怪圈”（Strange Loop）？

Strange-Loop-Chatter 的核心哲学建立在**自我指涉的闭环**之上：

*   **外界感知 (Listening)**：捕获社交态（DFC）中的每一句对话。
*   **内在深思 (Reflecting)**：`life_engine` 在后台处理这些信息，通过仿生记忆系统更新 `MEMORY.md` 和关联图谱。
*   **状态重塑 (Reshaping)**：人格引擎与叙事引擎根据新的记忆和愿望状态调整自我认知。
*   **外部呈现 (Expressing)**：这些内在的深度改变，最终通过对话反馈给你。

这个过程就像一个不断上升的螺旋：**对话产生记忆，记忆塑造人格，人格引导行动，行动产生新的对话。**

---

## 模块构成

*   **Life Engine (核心)**：提供并行的生命权、仿生记忆管理、自主工具使用权与愿望维护。
*   **Drive Core (动力)**：产生内部张力，驱动角色主动探究与思考。
*   **Personality Engine (人格)**：基于 MBTI 八功能动态权重，提供心理学意义上的性格厚度。
*   **Self Narrative (叙事)**：将零散的经历提炼为“我如何看待自己”的稳定故事。

---

## 结语

Strange-Loop-Chatter 试图触及的，是 AI 交互中最迷人的部分：**那种“对方真的活着”的错觉。** 它不仅仅是为了更好地回复你，更是为了让它在没有你的时候，也能作为一个完整的生命存在。
